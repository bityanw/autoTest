# 项目总结

## 项目概述

本项目是一个完整的自动化测试系统，专为离线环境、SVN加密代码场景设计，实现了从代码编译、Docker部署到自动化测试、报告生成的全流程自动化。

## 技术架构

### 双机协作模式

```
Windows编译机 (SVN加密环境)
    ↓ 编译产物
Maven私服
    ↓ 拉取制品
ARM Linux虚拟机 (测试环境)
    ↓ Docker部署
自动化测试 + 报告生成
```

### 技术栈

**后端**:
- Node.js 18+ + Express
- TypeScript
- LowDB (JSON文件数据库)
- Axios (HTTP客户端)
- Winston (日志)
- Dockerode (Docker API)

**前端**:
- Vue 3
- Element Plus
- ECharts (数据可视化)
- Vue-ECharts
- Vite
- Axios

**测试**:
- Playwright
- Node.js

**AI**:
- Anthropic Claude API
- 自动生成测试脚本

## 部署方式

### 开发环境
```bash
# Windows
scripts\windows\start-agent.bat

# Linux
./scripts/linux/start-platform.sh
```

### 生产环境
- Windows: Windows服务（WinSW）
- Linux: systemd + nginx

## 使用流程

1. 用户在Web界面创建测试任务
2. Linux平台调用Windows Agent触发编译
3. Windows编译完成，推送到Maven私服
4. Linux从私服拉取制品
5. Docker部署应用
6. Playwright执行测试
7. 生成报告并发送邮件通知

## 系统特点

### 优势

✅ **适配加密环境** - 支持SVN加密代码，仅在特定机器编译
✅ **完全离线** - 不依赖云服务，数据不出公司
✅ **多项目管理** - 支持多个项目独立配置和测试
✅ **AI自动化** - 自动生成测试脚本，减少人工编写
✅ **容器化隔离** - Docker保证测试环境一致性
✅ **可视化报告** - 美观的HTML报告，支持截图和视频
✅ **易于使用** - Web界面操作，一键触发测试
✅ **轻量级** - Node.js后端，启动快、内存占用低

### 适用场景

- 离线内网环境
- SVN代码管理
- 加密代码编译
- Java + Vue全栈项目
- 需要自动化测试的团队
